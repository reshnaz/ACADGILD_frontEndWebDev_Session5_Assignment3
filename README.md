# S5A3

## Problem Statement
1. Create a new folder and name it login. Create HTML, TS and CSS file in the
folder.
2. Create a component class LoginComponent and decorate it with
@Component().
3. In HTML file add Input field and Button and refer that HTML page in
@Component().
4. In CSS file add styles for the input and button and refer that CSS file in
@Component().
5. Add the selector of the LoginComponent to the RootComponent HTML
page.
